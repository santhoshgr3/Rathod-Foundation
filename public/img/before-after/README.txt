Drop the real BEFORE / AFTER photos here.

Expected filenames (referenced in src/data/content.ts):

  drain-before.jpg    / drain-after.jpg
  light-before.jpg    / light-after.jpg
  road-before.jpg     / road-after.jpg
  water-before.jpg    / water-after.jpg
  park-before.jpg     / park-after.jpg
  garbage-before.jpg  / garbage-after.jpg

Tips:
- Use the SAME aspect ratio for a before and its after (e.g. both 1200x800)
  so the drag-slider lines up cleanly.
- You can change any filename — just update the matching `before`/`after`
  string in src/data/content.ts.
- Until a file exists, the site shows a tidy placeholder, so nothing breaks.
